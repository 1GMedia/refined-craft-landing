# Refined Craft Landing Page Variants

Three distinct landing page designs for Refined Craft, each targeting different audiences while maintaining consistent core messaging.

## Variant A: Editorial Luxury
**Location:** `variant-a-editorial/`

**Target Audience:** Design-conscious professionals, premium learners

**Aesthetic:**
- Serif typography (Fraunces) with monospace accents
- Muted, sophisticated color palette
- Magazine-style editorial layout
- Generous whitespace
- Elegant, refined feel

**Best For:** Users who value craft, quality, and aesthetic sophistication

---

## Variant B: Brutalist Minimal
**Location:** `variant-b-brutalist/`

**Target Audience:** Developers, technical users, CLI enthusiasts

**Aesthetic:**
- Monospace typography (Space Mono)
- Black/white/neon green (#00FF00)
- Terminal/CLI-inspired design
- Raw, minimal layout
- Technical, no-nonsense feel

**Key Features:**
- ASCII art logo
- Terminal-style demo
- Bracket notation and CLI commands
- Zero decoration, maximum clarity

**Best For:** Developers who appreciate minimalism and terminal aesthetics

---

## Variant C: Soft Gradient
**Location:** `variant-c-gradient/`

**Target Audience:** General learners, non-technical users, approachable market

**Aesthetic:**
- Rounded sans-serif typography (DM Sans)
- Purple-to-orange gradient background
- Soft shadows and rounded corners
- Organic shapes and floating elements
- Warm, approachable feel

**Key Features:**
- Animated chat interface
- Floating organic shapes
- Soft color transitions
- Gentle, welcoming design language

**Best For:** General users who want an inviting, friendly learning experience

---

## Common Features (All Variants)

All three variants include:
- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations
- Interactive elements
- Same core messaging and value propositions
- Waitlist signup form
- Feature comparison table
- Use case examples

## Viewing the Variants

Each variant is self-contained with its own `index.html` and `styles.css`. Open any variant's `index.html` file in a browser to view.

```bash
# Variant A
open variant-a-editorial/index.html

# Variant B
open variant-b-brutalist/index.html

# Variant C
open variant-c-gradient/index.html
```

## A/B Testing Recommendations

1. **Split by Traffic Source:**
   - Product Hunt / Hacker News → Variant B (technical audience)
   - Design community / Dribbble → Variant A (design-conscious)
   - General social media → Variant C (broad appeal)

2. **Test Conversions:**
   - Track email signups per variant
   - Monitor time on page
   - Measure scroll depth

3. **User Feedback:**
   - Survey which aesthetic resonates
   - Ask about clarity of messaging
   - Test ease of understanding value prop
